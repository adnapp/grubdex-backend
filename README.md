Before setting up the frontend, install the database within the terminal of the root directory.:

$ rails db:create 
$ rails db:migrate 
$ rails db:seed 

Run the server:

$ rails s


Then proceed to start the <a href="https://github.com/adnapp/grubdex-frontend">frontend</a>.
